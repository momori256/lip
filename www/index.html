lip (in (Rust) (mini Lisp))

    Example

    • Literal (T = true, F = false)

      T

      => true

    • Logical operations (not, and, or)

      (^ (& T (| F F T)))

      => false (!(true & (false | false | true)))

    • Branching

      (if T & |)

      => primitive operator: &

    • Define variable

      (def nand (lambda (a b) (^ (& a b))))

      => lambda: (a b) -> (^ (& a b))

      (nand T T)

      => false
